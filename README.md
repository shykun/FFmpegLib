# FFmpegLib

Library for editing video using FFmpeg. With this library you are able perfom next operations: 
* Trim video
* Split video
* Reverse video
* Resize video
* Slowmotion
* Fastmotion
* Add text on video 
* Merge video and audio
* Extract audio from video
* Merge audio
* Trim audio 
* Convert video to gif
* Convert video to images
